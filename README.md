# Eight Puzzle Solver

  - A web app that solves the eight puzzle problem with multiple tree search algorithms, the tree search algorithms are implemented in C++ and the frontend of the app is implemented in ReactJS with Material UI, while Go acts as the backend by serving the static files from the ReactJS frontend and executes the tree search algorithm through the command line and returns the results so that the frontend can display it.
  - Search Algorithms Implemented
    - Depth First Search
    - Iterative Deepening Search
    - A* Search (Misplaced Tiles Heuristic)
    - A* Search (Manhattan Heuristic)
  - Available at [https://octetpuzzlesolver.herokuapp.com/]
